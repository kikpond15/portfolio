Code to Frame

Code to Frame is concept to frame a graphic being generated by program code
Artwork / Exhibition 2022

About

Code to Frameは、プログラム(Code)によって作り出された動的なデジタルアートを 静的な絵画のように額縁へプリントするためのフレームワークと、その展示形態を指します。 Code to Frameの展示形態は、あえて動的なデジタルアートの1フレームを切り取り、 鑑賞者へ静的な額縁として提示します。鑑賞者は額縁下部のQRコードを読み取る事で、 そのフレームに隠された動的なデジタルアートを自らの意思で鑑賞しに行くことが出来ます。
